Common questions

GTM shows ✓ Found but GA4 / Google Ads shows ✗ Missing

GTM is installed, but the tags inside it aren't firing on this page. Check your trigger conditions in GTM — they may not be matching this URL.

Google Ads shows "Tag in GTM — no ping seen" — is that a problem?

The tag exists in GTM but didn't fire on page load. Usually the trigger isn't set up correctly, or you're browsing in GTM's Preview mode which can affect tag firing.

GA4 shows "Tag loaded — no hit captured yet"

GA4's script is present but no measurement hit was recorded. Can happen with ad blockers, or if the GA4 Configuration tag trigger is set to something other than "All Pages".

Does this work on password-protected pages?

Yes. Unlike URL-based checkers, this runs in your browser so it works on any page you can view — including pages behind a login.

Does it send any of my data anywhere?

No. Everything runs locally in your browser. Nothing is sent to Teach Traffic's servers.

The bookmarklet does nothing when I click it

First try the "Test on this page" button above to confirm the code works. If that works, the issue is how the bookmark was saved — use the Manual Install option: copy the code, right-click your bookmarks bar, choose Add Bookmark, and paste the code into the URL field.
